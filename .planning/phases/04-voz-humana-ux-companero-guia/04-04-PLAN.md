<objective>
Capstone de Phase 4 — el diferenciador técnico. Cuando el usuario pregunta *"¿cómo mando un audio?"* o *"¿cómo hago una videollamada?"*, Beto:

1. El LLM (Phase 3) recibe la pregunta y emite la tool `show_how_to(action)`.
2. `GuideController` toma el control: abre la app target si no está, espera a que la pantalla cargue.
3. Por cada paso del script: localiza el `View` target vía `BetoAccessibilityService.findNodeByText/ContentDescription`, obtiene sus coordenadas, **dibuja una flecha animada apuntando a esas coordenadas** vía `GestureOverlay`, y dice el paso por TTS.
4. Espera 3 segundos antes del próximo paso para que el user procese.

V1 cubre 5 acciones cerradas con scripts curados. v2 será generativo.
</objective>

<user_story>
Como adulto mayor que no sabe cómo mandar un audio por WhatsApp, quiero preguntarle a Beto y que él me **muestre con una flecha en la pantalla** dónde apretar mientras me explica con la voz. Como si me lo enseñara mi nieto pero a cualquier hora.
</user_story>

<must_haves>
- GUIDE-01: Tool `show_how_to(action: enum)` definida en `ToolDescriptors`. Acciones v1: `send_whatsapp_audio`, `make_video_call`, `add_contact`, `increase_volume`, `open_camera`.
- GUIDE-02: `GestureOverlay` es un View añadido vía `WindowManager` (similar a `OverlayBubble` pero con `FLAG_NOT_FOCUSABLE | FLAG_NOT_TOUCHABLE`) que dibuja una flecha animada en coordenadas absolutas de pantalla. Animación: pulse + bounce vertical.
- GUIDE-03: `GuideScripts.kt` define para cada acción: lista de `GuideStep(targetSelector, voiceText, beforeAction)` donde `targetSelector` es texto o contentDescription a buscar.
- `BetoAccessibilityService.findNodeByText(text): AccessibilityNodeInfo?` y `findNodeByContentDescription(desc): AccessibilityNodeInfo?` busca DFS sobre `rootInActiveWindow` con cap 200 nodos.
- Si no se encuentra el target después de 2s + 1 retry: TTS warm error + abort guide.
- Si la app target no está en foreground, abrir vía Intent y esperar `WindowStateChanged` antes de continuar.
- Burbuja durante guía: estado `THINKING` durante búsqueda + transición a `SPEAKING` durante cada paso TTS. No se introduce sexto estado.
- User puede cancelar en cualquier momento tap a la burbuja → emite `GuideCancelled` → `GuideController` finaliza limpio.
</must_haves>

<threat_model>
| Threat | Severity | Mitigation |
|---|---|---|
| Flecha apunta al lugar incorrecto (target wrongly identified) | high | Selector específico (texto exacto o contentDescription oficial WhatsApp/Maps). Tests instrumented por acción en device pre-demo. |
| Flecha persistente si el guide crashea | high | `GestureOverlayManager` tiene cleanup explícito en finally + listener de `Lifecycle.onDestroy`. |
| App target cambia layout entre versiones (selector desactualizado) | high | Selectores múltiples por paso (try contentDescription primero, luego text). v1 documenta versión de WhatsApp testeada. |
| Coordenadas off-screen porque el View está scrolled | medium | Antes de pintar, verificar `getBoundsInScreen` está dentro de pantalla. Si no, scroll suave o abort con TTS. |
| Permission de overlay se desactiva durante guía | medium | Pre-check al inicio. Si no hay overlay perm, abort con TTS warm. |
| Animación de flecha lag en device modesto | medium | Usar ObjectAnimator simple, no Lottie. |
| User toca debajo de la flecha y dispara acción no deseada | low | `GestureOverlay` usa `FLAG_NOT_TOUCHABLE` — los toques pasan a la app debajo. Esto es intencional: queremos que el user toque el target real. |
</threat_model>

fun findNodeByText(text: String, root: AccessibilityNodeInfo? = rootInActiveWindow): AccessibilityNodeInfo? {
    if (root == null) return null
    var found: AccessibilityNodeInfo? = null
    bfs(root, cap = 200) { node ->
        if (node.text?.toString()?.equals(text, ignoreCase = true) == true ||
            node.text?.toString()?.contains(text, ignoreCase = true) == true) {
            found = node
            return@bfs true // stop
        }
        false
    }
    return found
}

fun findNodeByContentDescription(desc: String, root: AccessibilityNodeInfo? = rootInActiveWindow): AccessibilityNodeInfo? {
    // similar pero contra `node.contentDescription`
}

private fun bfs(root: AccessibilityNodeInfo, cap: Int, visit: (AccessibilityNodeInfo) -> Boolean) {
    val queue = ArrayDeque<AccessibilityNodeInfo>().apply { add(root) }
    var visited = 0
    while (queue.isNotEmpty() && visited < cap) {
        val n = queue.removeFirst(); visited++
        if (visit(n)) return
        for (i in 0 until n.childCount) n.getChild(i)?.let { queue.add(it) }
    }
}

fun nodeBoundsInScreen(node: AccessibilityNodeInfo): Rect = Rect().also { node.getBoundsInScreen(it) }
```

Exponer la instancia singleton vía un `BetoAccessibilityService.instance: BetoAccessibilityService?` que se setea en `onServiceConnected` y nullea en `onUnbind`.

Phase 1 ya creó el esqueleto, Phase 4 lo activa. Sin cambios al manifest.

class GestureOverlayManager(
    private val windowManager: WindowManager,
    private val context: Context,
) {
    private var overlay: GestureOverlay? = null

    fun show(targetBounds: Rect) {
        if (overlay == null) {
            overlay = GestureOverlay(context)
            val params = WindowManager.LayoutParams(
                WindowManager.LayoutParams.WRAP_CONTENT,
                WindowManager.LayoutParams.WRAP_CONTENT,
                WindowManager.LayoutParams.TYPE_ACCESSIBILITY_OVERLAY,
                WindowManager.LayoutParams.FLAG_NOT_FOCUSABLE
                    or WindowManager.LayoutParams.FLAG_NOT_TOUCHABLE
                    or WindowManager.LayoutParams.FLAG_LAYOUT_NO_LIMITS,
                PixelFormat.TRANSLUCENT
            )
            windowManager.addView(overlay, params)
        }
        overlay?.pointTo(targetBounds)
    }

    fun hide() {
        overlay?.let { runCatching { windowManager.removeView(it) } }
        overlay = null
    }
}
```

`GestureOverlay.pointTo(targetBounds: Rect)` actualiza posición y orientación de la flecha (apunta hacia el target desde arriba si hay espacio, sino desde abajo).

<title>GuideController</title>
<action>
`guide/GuideController.kt`:

```kotlin
class GuideController(
    private val accessibilityService: () -> BetoAccessibilityService?,
    private val gestureOverlay: GestureOverlayManager,
    private val tts: TtsManager,
    private val context: Context,
) {
    private var activeJob: Job? = null

    fun start(action: GuideAction, scope: CoroutineScope) {
        cancel()
        activeJob = scope.launch { runScript(action) }
    }

    fun cancel() {
        activeJob?.cancel()
        gestureOverlay.hide()
    }

    private suspend fun runScript(action: GuideAction) {
        val script = GuideScripts.forAction(action)
        val service = accessibilityService() ?: run {
            tts.speakAndAwait("No tengo permiso de accesibilidad. No puedo guiarte ahora.")
            return
        }

        try {
            if (script.appPackage != null && !isInForeground(script.appPackage)) {
                openApp(script.appPackage)
                delay(1500) // wait for screen
            }

            for (step in script.steps) {
                val node = findTargetWithRetry(service, step.targetSelector)
                if (node == null) {
                    tts.speakAndAwait("No encuentro lo que buscás. Probá abrir la app primero.")
                    return
                }
                val bounds = service.nodeBoundsInScreen(node)
                gestureOverlay.show(bounds)
                AgentBus.emit(AgentEvent.GuideStepShown)
                tts.speakAndAwait(step.voiceText)
                delay(3000)
                gestureOverlay.hide()
            }
            tts.speakAndAwait("Listo, eso es todo. Probá vos.")
        } finally {
            gestureOverlay.hide()
            AgentBus.emit(AgentEvent.GuideEnded)
        }
    }

    private suspend fun findTargetWithRetry(
        service: BetoAccessibilityService,
        selector: TargetSelector,
    ): AccessibilityNodeInfo? {
        repeat(2) {
            val node = when (selector) {
                is TargetSelector.ByText -> service.findNodeByText(selector.text)
                is TargetSelector.ByContentDescription -> service.findNodeByContentDescription(selector.desc)
            }
            if (node != null) return node
            delay(1500)
        }
        return null
    }
    // ...
}
```
