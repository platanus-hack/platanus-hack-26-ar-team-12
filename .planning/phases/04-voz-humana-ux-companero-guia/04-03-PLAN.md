<objective>
Implementar el Modo Compañero: long-press en la burbuja abre `CompanionActivity` (transparente con BottomSheet Compose) donde el usuario charla con Beto en español argentino cálido. Gemini 2.5 Flash Lite con `temperature: 0.4` para tono natural; modelo distinto al Motor de Acciones para no interferir con tool calling determinístico. Historial in-memory en ViewModel — se descarta al cerrar sheet por privacidad. Si el user dice algo profile-relevant (*"me gusta el tango"*, *"mi cumpleaños es en julio"*), Beto lo confirma y guarda en `UserMemoryStore` con `recordFact`.
</objective>

<user_story>
Como adulto mayor que vive solo, quiero poder charlar con Beto cuando quiera — preguntarle cosas, contarle cómo me siento, escuchar una respuesta cálida. No solo darle órdenes.
</user_story>

<threat_model>
| Threat | Severity | Mitigation |
|---|---|---|
| Sensitive content del Compañero filtrado a logs | high | Logs `Beto-Companion` solo loguean conteos / metadata, nunca contenido. |
| LLM responde con tono frío o formal | medium | System prompt explícito + few-shots argentinos + tests de regression con muestras. |
| Captura de facts inventa cosas que el user no dijo | high | El classifier usa `temperature: 0` y devuelve `null` si no hay fact claro. User confirma explícito con botón Sí — opt-in. |
| Sheet tapa flujo del Motor de Acciones (user da orden, no charla) | medium | Documentar separación: si user dice frase tipo comando, Beto sugiere *"¿Querés que ejecute X? Cerralo y tocá la burbuja."*. v1 deja la decisión al user. |
| Historial gigante satura memoria | low | Cap a últimos 20 mensajes en ViewModel. Mensajes más viejos se descartan. |
| Long-press accidental en arrastre rompe drag de la burbuja | medium | `OverlayBubble` distingue: long-press solo cuando el touch fue estacionario (movimiento total <8dp en 600ms). |
</threat_model>

```kotlin
class CompanionViewModel(
    private val llm: CompanionLlmClient,
    private val tts: TtsManager,
    private val voiceCapture: SuspendableVoiceCapture,
    private val memory: UserMemoryStore,
) : ViewModel() {

    private val _messages = MutableStateFlow<List<CompanionMessage>>(emptyList())
    val messages: StateFlow<List<CompanionMessage>> = _messages.asStateFlow()

    fun sendUserText(text: String) = viewModelScope.launch {
        appendUser(text)
        val reply = llm.chat(_messages.value)
        appendBeto(reply)
        tts.speakAndAwait(reply)

        val fact = llm.extractProfileFact(text)
        if (fact != null) attachFactConfirmation(fact)
    }

    fun startVoiceInput() = viewModelScope.launch {
        val captured = voiceCapture.captureOnce(15_000) ?: return@launch
        sendUserText(captured)
    }

    fun confirmFact(fact: ProfileFact, confirmed: Boolean) = viewModelScope.launch {
        if (confirmed) memory.recordFact(fact.category, fact.fact)
        clearFactConfirmation(fact)
    }

    fun forgetSession() {
        _messages.value = emptyList()
    }

    private fun appendUser(text: String) {
        val capped = (_messages.value + CompanionMessage(role = Role.USER, text = text)).takeLast(20)
        _messages.value = capped
    }
    // ...
}
```

```kotlin
class CompanionLlmClient(
    private val sanitizer: Sanitizer = Sanitizer,
) {
    private val chatModel: GenerativeModel = Firebase.ai(backend = GenerativeBackend.googleAI())
        .generativeModel(
            modelName = "gemini-2.5-flash-lite",
            generationConfig = generationConfig {
                temperature = 0.4f
                maxOutputTokens = 200
            },
        )

    private val factModel: GenerativeModel = Firebase.ai(backend = GenerativeBackend.googleAI())
        .generativeModel(
            modelName = "gemini-2.5-flash-lite",
            generationConfig = generationConfig {
                temperature = 0f
                maxOutputTokens = 80
                responseMimeType = "application/json"
            },
        )

    suspend fun chat(history: List<CompanionMessage>): String {
        val sanitized = history.map { it.copy(text = sanitizer.sanitize(it.text)) }
        val prompt = PromptBuilder.buildCompanionChat(sanitized)
        return chatModel.generateContent(prompt).text.orEmpty().ifBlank {
            "Estoy acá. Contame un poco más."
        }
    }

    suspend fun extractProfileFact(userText: String): ProfileFact? {
        val sanitized = sanitizer.sanitize(userText)
        val prompt = PromptBuilder.buildFactExtraction(sanitized)
        val raw = factModel.generateContent(prompt).text.orEmpty()
        return parseFactJson(raw) // null if no clear fact
    }
}
```

`PromptBuilder.buildCompanionChat`:
- System prompt: *"Sos Beto. Estás charlando con un adulto mayor en Argentina. Vocabulario simple. Voseo. Respondé en una o dos oraciones máximo. Sé cálido y paciente. NO ofrezcas hacer tareas — esto es solo charlar. Si el usuario dice algo personal sobre sus gustos, familia, hobbies, ciudad, no hagas comentarios largos."*
- Few-shots: 4 ejemplos de turnos cortos.

`PromptBuilder.buildFactExtraction`:
- System: *"Extraé un único hecho personal del mensaje si está claro y declarativo. Categorías: hobby, familia, ciudad, cumpleaños, mascota, otro. Si no hay fact claro, devolvé null."*
- Output schema: `{"fact": {"category": "...", "fact": "..."}|null}`.
- Ejemplos: *"Me gusta mucho el tango"* → `{"category":"hobby","fact":"tango"}`. *"Hola Beto"* → `null`.

```kotlin
@OptIn(ExperimentalMaterial3Api::class)
@Composable
fun CompanionSheet(
    onDismiss: () -> Unit,
    viewModel: CompanionViewModel = viewModel(),
) {
    val messages by viewModel.messages.collectAsState()
    val sheetState = rememberModalBottomSheetState(skipPartiallyExpanded = false)

    ModalBottomSheet(
        onDismissRequest = onDismiss,
        sheetState = sheetState,
    ) {
        Column(modifier = Modifier.fillMaxWidth().padding(16.dp)) {
            Text(
                "Estoy acá para charlar",
                style = MaterialTheme.typography.headlineLarge,
            )
            Spacer(Modifier.height(8.dp))
            LazyColumn(modifier = Modifier.weight(1f)) {
                items(messages) { msg ->
                    MessageRow(msg, onConfirmFact = viewModel::confirmFact)
                }
            }
            Row {
                BigButton(
                    text = "Hablar",
                    onClick = { viewModel.startVoiceInput() },
                )
                Spacer(Modifier.width(12.dp))
                BigButton(
                    text = "Olvidar charla",
                    onClick = { viewModel.forgetSession() },
                    secondary = true,
                )
            }
        }
    }
}
```

`MessageRow` muestra el texto con tipografía body 22sp + bubble visual diferente para USER vs BETO. Si el msg tiene `pendingFactConfirmation`, sumar tarjeta *"¿Querés que me acuerde de esto?"* con dos botones grandes Sí / No.
