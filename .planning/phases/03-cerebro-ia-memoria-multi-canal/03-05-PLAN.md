TTS pregunta *"¿Quién es tu nieto?"* → captura voz → busca con `ContactRepository.resolve()` → si match único, guarda `recordAlias("nieto", ContactRef)`; si match múltiple, vuelve a preguntar *"¿Cuál? - Plan C (matcher determinista) **mantiene precedencia**: si matchea, no se invoca el LLM. Sigue garantizando el guion principal sin red.
- Si la red cae durante un comando que normalmente iría al LLM, el dispatcher cae al matcher; si tampoco matchea, frase cálida *"No te entendí del todo, repetímelo más despacito."*.
- Todos los marcadores de log estables: `DISPATCH_START`, `DISPATCH_PLANC_HIT`, `DISPATCH_LLM_DECISION tool=X`, `DISPATCH_CLARIFY_CONTACT`, `DISPATCH_CLARIFY_CHANNEL`, `DISPATCH_EXECUTED`, `DISPATCH_FAILED reason=Y`.
</must_haves>

<threat_model>
| Threat | Severity | Mitigation |
|---|---|---|
| Bucle de clarificadores (Beto pregunta indefinidamente) | high | Cada clarificador tiene `MAX_ATTEMPTS = 3` (alineado con Phase 4 vieja); después de 3, *"Mejor lo intentamos en otro momento."* y vuelve a idle. |
| Mensaje enviado al contacto equivocado por homonimia silenciosa | high | Si `ContactRepository.resolve()` devuelve > 1 match, **siempre** preguntar al user — nunca asumir. |
| Aprende un alias incorrecto (user dice nombre mal) | medium | El clarificador resuelve contra `ContactRepository`; si no hay match, no guarda. Beto puede preguntar de nuevo o sugerir nombres parecidos. |
| Channel preference se aprende mal por audio confuso | medium | El matcher de canal acepta solo 3 keywords (`whatsapp`, `sms`, `llamada`/`telefono`). Si no matchea, repetir pregunta. |
| LLM tool call con args vacíos | medium | `ActionDispatcher` valida args no vacíos; si vacío, fallback a clarificación. |
| Latencia total > 10s desde tap a acción | medium | LLM cache (Plan 03-01) + matcher determinista + clarificadores cortos mantienen p95 < 10s para flujo típico. |
</threat_model>

```kotlin
fun makeCall(phoneE164: String): Boolean {
    val intent = Intent(Intent.ACTION_CALL, Uri.parse("tel:$phoneE164"))
        .addFlags(Intent.FLAG_ACTIVITY_NEW_TASK)
    return tryStart(intent, "make_call")
}

fun sendSms(phoneE164: String, message: String): Boolean {
    val uri = Uri.parse("smsto:$phoneE164")
    val intent = Intent(Intent.ACTION_SENDTO, uri).apply {
        putExtra("sms_body", message)
        addFlags(Intent.FLAG_ACTIVITY_NEW_TASK)
    }
    return tryStart(intent, "send_sms")
}

fun openMaps(query: String): Boolean {
    val uri = Uri.parse("geo:0,0?q=${Uri.encode(query)}")
    val intent = Intent(Intent.ACTION_VIEW, uri).apply {
        addFlags(Intent.FLAG_ACTIVITY_NEW_TASK)
    }
    return tryStart(intent, "open_maps")
}
```

```kotlin
class ContactClarifier(
    private val tts: TtsManager,
    private val contacts: ContactRepository,
    private val memory: UserMemoryStore,
    private val voiceCapture: SuspendableVoiceCapture, // wrapper suspend del VoiceCaptureActivity
) {
    suspend fun clarify(alias: String): ContactRef? {
        repeat(MAX_ATTEMPTS) { attempt ->
            tts.speakAndAwait("¿Quién es tu $alias?")
            val response = voiceCapture.captureOnce(timeoutMs = 10_000) ?: return null
            val matches = contacts.resolve(response)
            when (matches.size) {
                0 -> tts.speakAndAwait("No encuentro a nadie así. Probá con otro nombre.")
                1 -> {
                    val contact = matches[0]
                    val ref = contact.toContactRef()
                    memory.recordAlias(alias, ref)
                    Timber.tag("Beto-Action").d("DISPATCH_CLARIFY_CONTACT alias=$alias resolved=${ref.id}")
                    return ref
                }
                else -> {
                    val names = matches.joinToString(" o ") { it.displayName }
                    tts.speakAndAwait("Tengo varios. ¿Cuál? $names")
                    val pick = voiceCapture.captureOnce(timeoutMs = 10_000) ?: continue
                    val resolved = matches.firstOrNull { it.displayName.contains(pick, ignoreCase = true) }
                    if (resolved != null) {
                        val ref = resolved.toContactRef()
                        memory.recordAlias(alias, ref)
                        return ref
                    }
                }
            }
        }
        tts.speakAndAwait("Mejor lo intentamos en otro momento.")
        return null
    }
    companion object { const val MAX_ATTEMPTS = 3 }
}
```

`SuspendableVoiceCapture` es un nuevo helper que envuelve `VoiceCaptureActivity` lanzado y suspende hasta `VoiceCaptured` o timeout.

```kotlin
class ChannelClarifier(
    private val tts: TtsManager,
    private val voiceCapture: SuspendableVoiceCapture,
    private val memory: UserMemoryStore,
) {
    suspend fun clarify(contact: ContactRef): Channel? {
        repeat(MAX_ATTEMPTS) {
            tts.speakAndAwait("¿Por WhatsApp, SMS o llamada?")
            val r = voiceCapture.captureOnce(8_000)?.lowercase().orEmpty()
            val channel = when {
                "whatsapp" in r || "wasap" in r -> Channel.WHATSAPP
                "sms" in r || "mensaje" in r -> Channel.SMS
                "llamad" in r || "telefon" in r || "llamar" in r -> Channel.CALL
                else -> null
            }
            if (channel != null) {
                memory.recordChannel(contact, channel)
                Timber.tag("Beto-Action").d("DISPATCH_CLARIFY_CHANNEL contact=${contact.id} channel=$channel")
                return channel
            }
            tts.speakAndAwait("Decime por dónde — WhatsApp, mensaje o llamada.")
        }
        return null
    }
    companion object { const val MAX_ATTEMPTS = 2 }
}
```

```kotlin
class ActionDispatcher(
    private val matcher: DeterministicMatcher,
    private val llm: LlmClient,
    private val memory: UserMemoryStore,
    private val contacts: ContactRepository,
    private val contactClarifier: ContactClarifier,
    private val channelClarifier: ChannelClarifier,
    private val intentBranch: IntentBranch,
    private val tts: TtsManager,
    private val scope: CoroutineScope,
) {
    fun start() {
        AgentBus.events.filterIsInstance<AgentEvent.VoiceCaptured>()
            .onEach { handle(it.text) }
            .launchIn(scope)
    }

    private suspend fun handle(transcript: String) {
        Timber.tag("Beto-Action").d("DISPATCH_START")

        // 1. Plan C precedence
        matcher.match(transcript)?.let { match ->
            Timber.tag("Beto-Action").d("DISPATCH_PLANC_HIT")
            executeMatch(match); return
        }

        // 2. LLM
        val decision = runCatching { llm.interpret(transcript) }
            .getOrElse { Decision.Unknown }

        when (decision) {
            is Decision.ToolCall -> executeTool(decision)
            is Decision.NeedsClarification -> handleClarification(decision)
            Decision.Unknown -> tts.speak("No te entendí del todo, repetímelo más despacito.")
        }
    }

    private suspend fun executeTool(call: Decision.ToolCall) {
        val contact = resolveContactOrClarify(call.args["contact"]) ?: return

        val channel = inferChannelFromTool(call.tool)
            ?: memory.current().preferredChannel(contact)
            ?: channelClarifier.clarify(contact)
            ?: return

        val message = call.args["message"]
        // ... TTS confirmation, Intent launch, success/fail TTS
    }
    // ...
}
```
