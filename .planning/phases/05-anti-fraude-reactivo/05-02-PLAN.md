<objective>
Cablear las dos formas de invocar `FraudAnalyzer` desde el usuario:

1. **Por voz natural** — *"Beto, ¿esto es estafa?"* — el LLM router (Phase 3) reconoce la intención y emite tool `analyze_last_message`. `FraudController` toma el último SMS recibido (con permiso `READ_SMS`), lo manda al `FraudAnalyzer`, y habla el veredicto con voz neural masculina argentina.

2. **Por share-target** — el user comparte un screenshot o texto desde cualquier app, y elige Beto como destino. `FraudShareActivity` recibe el contenido, lo manda al analyzer, y habla el veredicto.

Si ninguna de las dos vías está disponible (sin permiso SMS, sin contenido compartido), Beto pide al user pegar/dictar el texto.
</objective>

<user_story>
Como adulto mayor, quiero pedirle a Beto que mire un mensaje sospechoso de la forma más natural posible — diciéndole que lo mire, o desde el botón compartir del celular — sin tener que copiar y pegar nada.
</user_story>

<must_haves>
- FRAUD-02: Tool `analyze_last_message` declarada para Gemini. El router reconoce frases como *"¿esto es estafa?"*, *"¿es real este mensaje?"*, *"mirá esto, ¿qué pensás?"*, *"me llegó algo raro, podés revisar?"*.
- `FraudController.analyzeLastSms()`: si `READ_SMS` permission, lee `Telephony.Sms.Inbox.CONTENT_URI` ordenado por `DATE DESC`, toma el más reciente recibido en últimas 2 horas, lo pasa al analyzer.
- Si no hay SMS reciente o no tiene permiso, `FraudController` dice por voz: *"No encontré ningún mensaje reciente para analizar. Mostrámelo o leémelo."*.
- `FraudShareActivity` registrada con `intent-filter` para `ACTION_SEND` con `text/plain` y `image/*`. Recibe el contenido, lo pasa al analyzer, habla veredicto, se cierra sola.
- Permission flow para `READ_SMS` opcional: si user lo rechaza, queda solo el path de share-target. NO bloqueante.
- Bubble state `THINKING` durante análisis (puede tomar hasta 8s); TTS de espera al inicio: *"Dejame leerlo un segundo..."*.
- TTS de veredicto consume `FraudVerdict.explanationWarm + suggestedAction` por el `PhraseGenerator` o directamente — preferir hablar el `explanationWarm` y luego el `suggestedAction` separados con pausa breve (más natural).
</must_haves>

<threat_model>
| Threat | Severity | Mitigation |
|---|---|---|
| Lectura de SMS no autorizada | high | Permission `READ_SMS` runtime con dialog estándar; degradación elegante si rechazado. |
| Beto analiza el SMS equivocado (no el que el user quería) | medium | Solo SMS recibidos en últimas 2h. Si hay > 1 SMS reciente, dice cuál: *"El último mensaje que te llegó es de [sender]. ¿Es ese?"* y espera confirmación antes de analizar. |
| Share-target abre la app sin permiso de overlay activo | medium | `FraudShareActivity` no requiere overlay — es Activity propia. Sin asunción de permisos extra. |
| User comparte algo enorme (libro entero) | low | Cap de 4000 chars al pasar a `FraudAnalyzer`. Truncar con frase warm. |
| Screenshot share llega como Uri inaccesible | medium | Try/catch + frase warm: *"No puedo abrir ese archivo. Mejor leémelo o pegámelo."*. |
| Crash de `FraudShareActivity` deja al user sin feedback | high | `try { } catch { speakAndAwait("Algo no salió bien, probemos de nuevo") } finally { finish() }`. |
</threat_model>

<title>FraudController con análisis de último SMS</title>
<action>
`fraud/FraudController.kt`:

```kotlin
class FraudController(
    private val context: Context,
    private val analyzer: FraudAnalyzer,
    private val tts: TtsManager,
    private val phraseGen: PhraseGenerator,
) {

    suspend fun analyzeLastSms(): FraudVerdict? {
        if (!hasSmsPermission()) {
            tts.speakAndAwait("No tengo acceso a tus mensajes. Mostrámelo apretando compartir.")
            return null
        }

        val recent = readLastInboxSms(maxAgeMs = TimeUnit.HOURS.toMillis(2))
        if (recent == null) {
            tts.speakAndAwait("No encontré mensajes recientes. Mostrámelo o leémelo.")
            return null
        }

        // Confirmation if multiple recent
        // (simplificado — Plan 05-02 v1 toma el más nuevo)
        tts.speakAndAwait("Dejame leerlo un segundo...")
        AgentBus.emit(AgentEvent.BubbleStateOverride(BubbleState.THINKING))

        val verdict = analyzer.analyze(recent.body)

        tts.speakAndAwait(verdict.explanationWarm)
        delay(400)
        tts.speakAndAwait(verdict.suggestedAction)
        AgentBus.emit(AgentEvent.BubbleStateOverride(BubbleState.IDLE))
        return verdict
    }

    suspend fun analyzeText(text: String, screenshot: Bitmap? = null): FraudVerdict {
        tts.speakAndAwait("Dejame revisarlo...")
        AgentBus.emit(AgentEvent.BubbleStateOverride(BubbleState.THINKING))
        val verdict = analyzer.analyze(text.take(4000), screenshot)
        tts.speakAndAwait(verdict.explanationWarm)
        delay(400)
        tts.speakAndAwait(verdict.suggestedAction)
        AgentBus.emit(AgentEvent.BubbleStateOverride(BubbleState.IDLE))
        return verdict
    }

    private fun hasSmsPermission() = context.checkSelfPermission(Manifest.permission.READ_SMS) == PackageManager.PERMISSION_GRANTED

    private fun readLastInboxSms(maxAgeMs: Long): SmsMessage? {
        val cursor = context.contentResolver.query(
            Telephony.Sms.Inbox.CONTENT_URI,
            arrayOf("_id", "address", "body", "date"),
            "date >= ?",
            arrayOf((System.currentTimeMillis() - maxAgeMs).toString()),
            "date DESC LIMIT 1",
        )
        return cursor?.use {
            if (it.moveToFirst()) {
                SmsMessage(
                    address = it.getString(it.getColumnIndexOrThrow("address")),
                    body = it.getString(it.getColumnIndexOrThrow("body")),
                    timestamp = it.getLong(it.getColumnIndexOrThrow("date")),
                )
            } else null
        }
    }

    data class SmsMessage(val address: String, val body: String, val timestamp: Long)
}
```
</action>

`fraud/FraudShareActivity.kt`:

```kotlin
class FraudShareActivity : ComponentActivity() {
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        lifecycleScope.launch {
            try {
                val (text, bitmap) = extractContent(intent)
                if (text.isNullOrBlank() && bitmap == null) {
                    Container.tts.speakAndAwait("No pude leer lo que compartiste. Probemos de nuevo.")
                    return@launch
                }
                Container.fraudController.analyzeText(text.orEmpty(), bitmap)
            } catch (e: Exception) {
                Timber.tag("Beto-Fraud").e(e, "Share crash")
                Container.tts.speakAndAwait("Algo no salió bien, probemos de nuevo.")
            } finally {
                finish()
            }
        }
    }

    private suspend fun extractContent(intent: Intent): Pair<String?, Bitmap?> {
        val text = intent.getStringExtra(Intent.EXTRA_TEXT)
        val uri = intent.getParcelableExtra<Uri>(Intent.EXTRA_STREAM)
        val bitmap = uri?.let { loadBitmap(it) }
        return text to bitmap
    }
}
```

`Container` es el holder de singletons — lo sumamos en `BetoApplication`.
