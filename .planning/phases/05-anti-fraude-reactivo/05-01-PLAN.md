<objective>
Implementar la lógica de análisis de fraude: tool `analyze_for_fraud(text, screenshot?)` declarada en `ToolDescriptors`, `FraudAnalyzer` que invoca Gemini con prompt curado y ejemplos few-shot de estafas argentinas reales, modelo de respuesta `FraudVerdict` estructurado, y wiring al `LlmClient`. La UX de invocación (cómo el user lo dispara) va en Plan 05-02.
</objective>

<user_story>
Como adulto mayor, cuando me llega un mensaje raro, quiero pedirle a Beto que lo analice y que me diga claramente si parece estafa, sin asustarme y con palabras simples.
</user_story>

<must_haves>
- FRAUD-01: `FraudAnalyzer.analyze(text: String, screenshot: Bitmap? = null): FraudVerdict`. Retorna `FraudVerdict(isFraud: Boolean, riskLevel: RiskLevel, explanationWarm: String, suggestedAction: String, signals: List<String>)`.
- `RiskLevel = LOW | MEDIUM | HIGH`. `LOW` = parece legítimo. `MEDIUM` = sospechoso. `HIGH` = casi seguro estafa.
- Prompt curado en es-AR con sistema explícito: identidad (*"Sos un experto en detectar estafas a adultos mayores en Argentina"*), patrones a buscar (urgencia, pedido de DNI/CBU/clave/código de seguridad, suplantación de organismos, premios falsos), tono de salida (cálido, sin alarmismo, voseo, ≤2 oraciones).
- 6+ few-shots reales: AFIP falsa, ANSES falsa, banco falso, premio falso, "tu nieto está en problema" (estafa del cuento del tío versión moderna), reset de password legítimo (control negativo).
- `Sanitizer.sanitize` aplicado al texto antes de pasarlo al LLM.
- Si screenshot está presente, se manda multimodal a Gemini Vision (mismo modelo `gemini-2.5-flash`, imagen + texto).
- Output schema validado con kotlinx.serialization. Si malformado, retry 1x con error inline; si segundo retry falla, devolver `FraudVerdict.unknown()` (riskLevel=MEDIUM, mensaje warm de fallback).
- Logs `Beto-Fraud`: `FRAUD_ANALYZE_START`, `FRAUD_VERDICT risk=$level isFraud=$bool signals=$count`.
</must_haves>

<threat_model>
| Threat | Severity | Mitigation |
|---|---|---|
| Falso negativo: Beto descarta una estafa real | high | Few-shots cubren patrones argentinos comunes; `riskLevel=LOW` exige al LLM justificar con señales positivas; cuando duda, defaultear a `MEDIUM`. |
| Falso positivo: Beto alerta sobre mensaje legítimo | high | Control negativo en few-shots (banco real con estilo formal). Prompt especifica que sin urgencia + sin pedido de datos sensibles → posiblemente legítimo. |
| Alarmismo asusta al user | high | Quality check sobre `explanationWarm`: descarta si contiene "PELIGRO", "URGENTE", caps lock, signos de exclamación múltiples. |
| Datos sensibles del mensaje filtrado se loguean | high | Sanitizer + nunca loguear `text` raw. Solo `len=N hasScreenshot=true/false`. |
| Modelo no reconoce screenshots de mensajería argentina (WhatsApp, SMS Android) | medium | Few-shots con descripciones de screenshots típicos. Probable que Gemini Vision lo maneje bien (mensajería universal). |
| Latencia de análisis > 8s rompe paciencia del user | medium | `BubbleState.THINKING` mientras espera + frase TTS de espera *"Dejame pensarlo un segundo..."*. Timeout 8s con fallback. |
</threat_model>

`fraud/FraudAnalyzer.kt`:

```kotlin
class FraudAnalyzer(private val llm: LlmClient) {
    suspend fun analyze(text: String, screenshot: Bitmap? = null): FraudVerdict {
        Timber.tag("Beto-Fraud").d("FRAUD_ANALYZE_START len=${text.length} hasScreenshot=${screenshot != null}")

        val raw = llm.analyzeForFraud(text, screenshot)
        if (raw.isBlank()) return FraudVerdict.unknown()

        val verdict = parseVerdict(raw)
            ?: parseVerdict(retryOnce(text, screenshot, raw))
            ?: return FraudVerdict.unknown()

        if (!passesQualityChecks(verdict)) return FraudVerdict.unknown()

        Timber.tag("Beto-Fraud").d("FRAUD_VERDICT risk=${verdict.riskLevel} isFraud=${verdict.isFraud}")
        return verdict
    }

    private fun passesQualityChecks(v: FraudVerdict): Boolean {
        val text = v.explanationWarm
        if (text.contains("PELIGRO", ignoreCase = true)) return false
        if (text.contains(Regex("!{2,}"))) return false
        if (text.split(" ").size > 30) return false  // demasiado largo
        return true
    }
    // ...
}
```
